## Data Collection
- This app does not collect, store, or share any user data.
- No analytics, advertising, or third-party SDKs are integrated.
- No account systems or network connections are used.

## Local Storage
- Any saves or preferences are stored locally on the player’s device using Unity’s built-in PlayerPrefs system.
- Data remains on-device and is never transmitted externally.

## Network Access
- The current version does not perform any network requests.

## Priacy Policy

This Privacy Policy describes how Final Keep collects, uses, and protects your information when you use our game and related services.

1. Introduction

Final Keep respects your privacy. This Privacy Policy explains what information (if any) our game collects, how it is used, and the limited ways it may interact with your device.

Final Keep is a standalone mobile game created as part of an academic module. It does not connect to online servers, collect personal information, or require account registration.

2. Information We Collect

Final Keep does not collect, store, or transmit any personal or identifying information.
No player accounts, sign-ins, or cloud saves are implemented in this version.

The only information handled by the app is:

Basic device performance data temporarily used by Unity’s runtime (e.g., framerate, memory usage).

Local gameplay data, such as player progress, stored only on your device and not shared externally.

3. How We Use the Information

Any temporary or local data is used solely to:

Run the game smoothly on your device

Save player progress between sessions (if applicable)

This data is not transmitted or shared with any third parties.

4. Data Storage and Security

All gameplay data remains on the player’s device within Unity’s local storage system.
Final Keep does not connect to any external databases, servers, or APIs.

Unity’s built-in systems apply standard Android sandboxing for app data, which prevents other apps from accessing it.

5. Advertising and Analytics
This version of Final Keep does not use any advertising or analytics SDKs (such as AdMob, Unity Analytics, or Firebase).

Future versions may include analytics or ads — if so, this policy will be updated to reflect those changes before any data is collected.

6. Third-Party Services and Links

Final Keep uses the Unity game engine to run on Android. Unity itself may process limited non-identifiable system data (e.g., device type or OS version) as part of its runtime.
For details, refer to Unity’s privacy documentation:
https://unity.com/legal/privacy-policy

7. Data Sharing

We do not share, sell, or transmit any personal information to other parties.
All gameplay and performance data remains local to your device.

8. Children’s Privacy

Final Keep is suitable for all ages and does not collect or process personal data.
Children can play safely without providing any personal information.

If your child uses Google Sign-In, only minimal account data (such as a unique identifier) is used to maintain game progress.

9. User Rights and Data Control

Since Final Keep does not collect or store user data externally, there is no personal information to access, modify, or delete.
You can uninstall the app at any time to remove all locally stored game data.

10. Changes to This Policy

If the game is updated to include online features, analytics, or ads, this Privacy Policy will be revised to describe any new data practices.
The latest version of this policy will always be available with the game files and repository.
