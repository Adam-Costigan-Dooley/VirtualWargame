Lineage Virtual Wargaming

Automating Discord Wargame Turn Resolution

About

Lineage Virtual Wargaming transforms multi-day Discord wargame turns into minutes of automated resolution. Built for tabletop gaming communities, this Unity-based system brings strategic depth to digital multiplayer gameplay.

Strategic Combat

1Territory map with sophisticated combat mechanics, casualty resolution, and unit special abilities.

Multiplayer

2-6 player support via Photon Fusion with real-time state synchronization.

Asymmetric Factions

6 unique factions with specialized units including heroes, villains, and scouts.

Custom Layouts

Host-configurable map layouts with team selection and resource allocation.

Technologies

Unity 2022 • C# • Photon Fusion 2 • Aseprite • Git
