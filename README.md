## Controls

# Mobile
- Move Left = Touch left area of screen
- Move Right = Touch right area of screen
- Build/Upgrade = Touch central screen region while within interaction trigger
- Pause	Touch = Pause button
- Quit to Menu = From Pause or Game Over screen

# Desktop
- Move Left = A
- Move Right = D
- Build/Upgrade = E
- Pause	Touch = Escape
